# substrate-guard v13.2.8 — Bijuteria #5 AST-first CLI safety

## Modul `ast_parse`

- **`parser.py`** — Tree-sitter bash (opțional prin deps), euristică `detect_shell_language`, unwrap `sudo` (`-u user`, flag-uri).
- **`patterns.py`** — `rm` + flag-uri `r`/`f`, `chmod 777`, `dd of=/dev/...`, `mkfs`, pipeline `curl|wget` → `sh`/`bash`, Python `eval`/`exec`/`compile`/`__import__` via `ast`.
- **`safety_checker.py`** — `check_shell_command_ast()`; limită dimensiune input.

## Integrare

- **`cli_verifier`**: verificări structurale **înainte** de pattern-urile regex existente; încălcările AST apar în `CLIViolation.pattern_name` cu prefix `ast_`.

## Dependențe

- **`tree-sitter`**, **`tree-sitter-bash`** adăugate la **`[project.optional-dependencies] dev`**.
